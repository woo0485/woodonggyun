프로젝트 명 : GALAGO

소개
- 저희 사이트는 다양한 축제 상품을 소개하고 직접 판매하는 플랫폼입니다.

개발 목표
- 무료와 유료 상품을 구별하여 제공함으로써 사용자들에게 편리함을 제공하고, 이를 통해 축제 참여자의 만족도를 높이고 수익을 창출합니다.
